# docs/ 인덱스

이 폴더는 프론트엔드·백엔드 개발에 필요한 가이드라인, 표준, 프로젝트 관리 템플릿을 담고 있습니다.

---

## 폴더 구조

```
docs/
├── README.md              ← 이 파일 (인덱스)
├── guidelines/            ← 개발 가이드라인 
├── standards/             ← 프론트엔드/백엔드 개발 표준
└── templates/             ← 프로젝트 관리 문서 템플릿
```
